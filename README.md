## ✨ Features

- ✅ **Rate limiting**
- ✅ **Auth**
- ✅ **Caching**
- ✅ **Load balancing**

## 🛠️ Tech Stack

- **Redis**
- **JWT**
- **OpenAPI**
- **FastAPI**

## 🚀 Quick Start

```bash
# Docker Compose (Recommended)
docker-compose up -d

# Local Development
pip install -r requirements.txt
python app/gateway.py
```

## 📖 Configuration

Edit `config/config.yaml` to customize the application.

## 🔗 API Endpoints

- `GET /health` - Health check
- `GET /` - Service information
